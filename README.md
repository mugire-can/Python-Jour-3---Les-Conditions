# Python Day 3 - Conditions (Jour 3 - Les Conditions)

A collection of Python exercises focusing on conditional statements and control flow from La Plateforme Python course.

## Overview

This project contains 8 programming exercises demonstrating fundamental Python concepts:

1. **Job 1**: Value comparison - Check if two input values are equal
2. **Job 2**: Voting eligibility - Determine if age meets voting requirements
3. **Job 3**: Loop with continue - Print numbers 0-100 except 26, 37, 88
4. **Job 4**: FizzBuzz - Classic programming problem with modulo operations
5. **Job 5**: Prime numbers - Find all prime numbers between 2-1000
6. **Job 6**: Even/odd checker - Determine if a number is even or odd
7. **Job 7**: String pyramid - Build a pyramid pattern from user input
8. **Job 8**: Triangle validator - Classify triangles by type (equilateral, isosceles, rectangle, scalene)

## Requirements

- Python 3.7+

## Usage

Run the main script:

```bash
python "Projet Jour 3.py"
```

The script contains multiple exercises that can be run interactively. Each job section includes input prompts for user data.

## Topics Covered

- Conditional statements (`if`, `elif`, `else`)
- Comparison operators (`==`, `!=`, `>`, `<`, `>=`, `<=`)
- Logical operators (`and`, `or`)
- Loops (`for`, `range`)
- Loop control (`continue`, `break`)
- Functions and return statements
- String slicing
- Mathematical operations
